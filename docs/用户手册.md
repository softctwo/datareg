## 1. 系统概述

### 1.1 系统简介

银行重要数据跨境数据管控系统是一个面向银行的数据出境合规管理平台，旨在帮助银行：

- 管理数据资产分类分级
- 控制跨境数据传输场景
- 进行数据出境风险评估
- 监控数据传输活动
- 确保合规审计要求

### 1.2 目标用户

本系统主要面向以下用户：

- **数据管理员**：负责数据资产管理和分类分级
- **合规专员**：负责跨境场景管理和风险评估
- **审批人员**：负责数据传输审批
- **审计人员**：负责监控和审计
- **系统管理员**：负责用户和权限管理

### 1.3 系统特性

- ✅ **全流程管控**：覆盖数据资产识别到传输监控的完整流程
- ✅ **智能识别**：自动发现和分类数据资产
- ✅ **风险评估**：内置PIA/DPIA评估模板
- ✅ **实时监控**：全程监控数据传输活动
- ✅ **合规保障**：符合国家法律法规要求

## 2. 快速开始

### 2.1 登录系统

#### 2.1.1 访问地址

打开浏览器，访问系统地址：
```

## 4. 功能模块使用指南

### 4.1 监控仪表盘

#### 4.1.1 功能概述

仪表盘提供系统整体运行情况的实时监控，包括：

- 场景统计
- 审批状态
- 传输趋势
- 风险预警
- 目的国分布

#### 4.1.2 使用方法

**访问仪表盘**
1. 点击左侧菜单"仪表盘"
2. 系统自动加载最新数据

**查看统计信息**
- 场景总数：显示系统中所有跨境场景数量

**查看传输趋势**
- 图表显示最近30天的传输量变化趋势
- 可以选择不同的时间范围（7天、30天、90天）
- 支持按场景类型筛选

**查看目的国分布**
- 饼图显示数据传输到各国家的分布情况
- 点击饼图区块可以查看详细信息

**查看风险预警**
- 显示当前活跃的风险预警信息
- 点击预警项可以查看详细信息和处理建议

### 4.2 数据资产管理

#### 4.2.1 功能概述

数据资产管理模块负责：

- 自动扫描数据库发现数据资产
- 对数据资产进行分类分级
- 管理敏感标签
- 维护数据血缘关系

#### 4.2.2 数据扫描

**启动自动扫描**
1. 进入"数据资产"页面
2. 点击"扫描数据"按钮
3. 配置数据库连接信息：
   - 数据库主机
   - 端口号
   - 数据库名
   - 用户名
   - 密码
4. 点击"开始扫描"

**扫描结果查看**
- 扫描完成后自动显示发现的数据表
- 每个表显示基本信息：表名、字段数、记录数
- 系统自动进行初步分类分级

#### 4.2.3 分类分级管理

**查看数据资产列表**
- 页面显示所有数据资产
- 可以按以下条件筛选：
  - 数据级别（核心/重要/敏感/个人/内部/公开）
  - 数据分类
  - 敏感标签
  - 扫描时间

**修改分类分级**
1. 在列表中选择要修改的数据资产
2. 点击"编辑"按钮
3. 在弹出的对话框中：
   - 修改数据级别
   - 选择数据分类
   - 添加敏感标签
4. 点击"保存"

**批量操作**
- 支持批量修改数据级别
- 支持批量添加标签
- 使用复选框选择多个资产进行操作

#### 4.2.4 血缘关系管理

**查看血缘关系**
1. 点击数据资产行的"血缘关系"按钮
2. 系统显示该资产的上游和下游依赖关系
3. 可以点击关联资产查看详情

**编辑血缘关系**
1. 在血缘关系页面点击"编辑"按钮
2. 添加或删除上下游关联
3. 保存修改

   └─────────────┘

### 4.3 跨境场景管理

#### 4.3.1 功能概述

跨境场景管理负责：

- 创建和管理跨境传输场景
- 定义传输要素和安全措施
- 管理场景生命周期
- 控制场景有效期

#### 4.3.2 创建场景

**基本信息填写**
1. 点击"新增场景"按钮
2. 填写场景基本信息：
   - 场景名称（必填）
   - 场景编码（系统自动生成）
   - 业务类型（审计/合规审查/报表汇总/其他）
   - 描述信息

**接收方信息**
- 接收方名称（必填）
- 接收方所在国（必填）
- 接收方类型（母行/境外分行/第三方机构）

**传输要素配置**
- 数据用途（必填）
- 存储期限（天）
- 传输频率（实时/日/周/月/一次性）
- 链路安全级别（高/中/低）
- 加密方式

**数据范围定义**
- 数据范围描述
- 预估数据量
- 涉及的数据资产（从数据资产列表中选择）

#### 4.3.3 场景审批

**提交审批**
1. 完成场景信息填写后
2. 点击"提交审批"按钮
3. 系统验证信息完整性
4. 提交成功后场景状态变为"待审批"

**查看审批状态**
- 场景列表显示审批状态
- 可以查看审批历史和意见
- 支持按状态筛选场景

#### 4.3.4 场景管理

**编辑场景**
- 只有草稿状态的场景可以编辑
- 已提交的场景不能修改，需要先撤回

**场景状态管理**
- **草稿**：可以编辑和删除
- **待审批**：可以撤回
- **已批准**：可以使用或暂停
- **已拒绝**：可以修改后重新提交
- **已过期**：需要重新申请
- **已暂停**：可以恢复使用

**有效期管理**
- 设置场景的有效期
- 系统自动检测过期场景
- 过期前7天发送提醒

   └─────────────┘

### 4.4 风险评估

#### 4.4.1 功能概述

风险评估模块提供：

- PIA/DPIA评估模板
- 多维度风险分析
- 阈值检查和预警
- 风险等级计算
- 评估报告生成

#### 4.4.2 创建评估

**选择评估类型**
1. 点击"新建评估"按钮
2. 选择评估类型：
   - PIA（个人信息影响评估）
   - DPIA（数据保护影响评估）
3. 选择关联的跨境场景

**填写评估信息**
- 评估名称
- 评估目的
- 评估范围

**多维度评分**
- **法律环境**（0-100分）：
  - 接收方所在国数据保护法律完善程度
  - 法律执行力度
  - 与我国法律的一致性
- **数据规模**（0-100分）：
  - 个人信息数量
  - 敏感个人信息比例
  - 数据更新频率
- **安全措施**（0-100分）：
  - 传输加密强度
  - 存储安全措施
  - 访问控制机制
- **数据敏感性**（0-100分）：
  - 数据安全级别
  - 敏感信息比例
  - 数据关联性

#### 4.4.3 阈值检查

系统自动检查以下阈值：

- **个人信息阈值**：100万条个人信息
- **敏感信息阈值**：10万条敏感个人信息

**预警机制**
- 达到阈值95%时显示黄色预警
- 超过阈值时显示红色警告
- 需要申报监管审批时特别提醒

#### 4.4.4 风险等级计算

系统根据各维度评分计算综合风险等级：

| 综合评分 | 风险等级 | 建议 |
|----------|----------|------|
| 80-100分 | 低风险 | 可以正常传输 |
| 60-79分 | 中风险 | 需要基本保护措施 |
| 40-59分 | 高风险 | 需要加强保护措施 |
| 0-39分 | 极高风险 | 不建议传输 |

#### 4.4.5 评估报告

**生成报告**
1. 完成评估填写
2. 点击"生成报告"按钮
3. 系统自动生成评估报告
4. 报告包含：
   - 评估概述
   - 风险分析
   - 缓解措施
   - 建议

**导出报告**
- 支持PDF格式导出
- 支持打印功能
- 可以分享给相关方

```

   ┌─────────────┐

   └─────────────┘
   ↓
   ┌─────────────┐
   │  阈值检查     │ 自动检查是否超阈值

### 4.5 传输审批

#### 4.5.1 功能概述

传输审批模块负责：

- 管理数据传输审批申请
- 执行审批流程
- 记录审批结果
- 追踪传输状态

#### 4.5.2 申请传输

**创建审批申请**
1. 点击"新建审批"按钮
2. 选择关联的跨境场景
3. 填写传输信息：
   - 传输类型（API/文件/数据库）
   - 传输目的
   - 预计传输时间
   - 预计传输量
4. 选择要传输的数据资产
5. 点击"提交申请"

**申请状态跟踪**
- **待审批**：等待审批人处理
- **已批准**：可以执行传输
- **已拒绝**：传输被拒绝，需要修改后重新申请
- **已取消**：申请人取消申请

#### 4.5.3 审批处理

**审批人操作**
1. 进入"传输审批"页面
2. 查看待审批申请列表
3. 点击申请编号查看详情
4. 审核申请内容：
   - 场景合规性
   - 数据范围
   - 传输必要性
5. 做出审批决定：
   - 批准：填写审批意见，点击"批准"
   - 拒绝：填写拒绝原因，点击"拒绝"

**批量审批**
- 支持批量批准符合条件的申请
- 批量操作时需要填写统一的审批意见

#### 4.5.4 传输记录

**记录传输信息**
- 传输开始时间
- 传输结束时间
- 实际传输量
- 传输状态（成功/失败/拦截）

**查询传输记录**
- 可以按申请人、时间段、状态等条件查询
- 支持导出传输记录

## 5. 常见问题解答

### 5.1 登录问题

**Q: 忘记密码怎么办？**
A: 联系系统管理员重置密码。

**Q: 登录时提示"用户名或密码错误"？**
A: 请检查：
- 用户名输入是否正确
- 密码大小写是否正确
- Caps Lock是否开启
- 输入法是否为英文状态

**Q: 登录后自动退出怎么办？**
A: 可能是会话超时，请重新登录。建议检查网络连接是否稳定。

### 5.2 数据扫描问题

**Q: 数据扫描失败怎么办？**
A: 请检查：
- 数据库连接信息是否正确
- 数据库服务是否正常运行
- 网络连接是否通畅
- 用户权限是否足够

**Q: 扫描结果不完整？**
A: 可能原因：
- 部分表没有访问权限
- 数据库连接超时
- 数据量过大导致超时

### 5.3 审批问题

**Q: 审批申请被拒绝了怎么办？**
A: 请：
1. 查看拒绝原因
2. 修改申请内容
3. 重新提交申请

**Q: 审批流程太慢怎么办？**
A: 可以：
1. 联系审批人催办
2. 联系系统管理员检查流程配置
3. 确认申请信息是否完整

### 5.4 系统性能

**Q: 页面加载缓慢怎么办？**
A: 建议尝试：
1. 清理浏览器缓存
2. 使用Chrome或Firefox浏览器
3. 检查网络连接质量
4. 联系系统管理员

### 5.5 其他问题

**Q: 如何导出数据？**
A: 大部分页面都支持导出功能：
1. 点击"导出"按钮
2. 选择导出格式（CSV/JSON/PDF）
3. 下载导出文件

**Q: 如何联系技术支持？**
A: 可以通过以下方式：
- 系统内帮助文档
- 邮件：support@bank.com
- 电话：400-XXX-XXXX
